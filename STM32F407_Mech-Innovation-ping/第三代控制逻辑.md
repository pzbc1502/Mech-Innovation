# 第三代三层洗菜系统控制逻辑规格

## 1. 目标与边界

本文档用于后续代码改造，描述第三代机械结构下的自动运行流程、状态机、电机编号、称重、丝杆推板、打包和停机逻辑。

本版本只定义控制逻辑，不修改业务源码。电机通信链路保持为：

```text
CAN2 -> Emm_V5 -> can_SendCmd
```

第三代结构中，第三层不再是普通下层传送带，而是：

```text
称重板 + 双 HX711 + 5/6号丝杆推板 + 7号旋转打包电机
```

8号电机在第三代结构中不参与自动流程，后续代码应从第三代控制流程中移除 8 号电机的启动、停止和状态调度。

## 2. 电机编号与执行机构

| 模块   | 编号 | 用途             | 控制模式 | 默认方向          | 默认速度 | 备注                           |
| ------ | ---- | ---------------- | -------- | ----------------- | -------- | ------------------------------ |
| 上层   | 1    | 上层传送带电机 1 | 速度模式 | CW                | 60 RPM   | 与 2 号同步                    |
| 上层   | 2    | 上层传送带电机 2 | 速度模式 | CW                | 60 RPM   | 与 1 号同步                    |
| 中层   | 3    | 中层传送带电机 1 | 速度模式 | CCW               | 60 RPM   | 与 4 号同步                    |
| 中层   | 4    | 中层传送带电机 2 | 速度模式 | CW                | 60 RPM   | 与 3 号同步                    |
| 第三层 | 5    | 丝杆推板电机 1   | 位置模式 | 推动 CCW，返回 CW | 30 RPM   | 与 6 号同步                    |
| 第三层 | 6    | 丝杆推板电机 2   | 位置模式 | 推动 CCW，返回 CW | 30 RPM   | 与 5 号同步                    |
| 打包   | 7    | 旋转打包电机     | 速度模式 | CCW               | 30 RPM   | 推动时同步启动，到位后继续运行 |
| 保留   | 8    | 第三代不使用     | 不参与   | 不参与            | 不参与   | 后续代码不调度                 |

执行机构：

| 名称        | 用途                       | 当前接口                                                    |
| ----------- | -------------------------- | ----------------------------------------------------------- |
| 切刀        | 中层切割                   | `RELAY_CUTTER`                                            |
| 水泵        | 如后续需要清洗水路，可保留 | `RELAY_PUMP`                                              |
| 下层舵机 | 放料或挡料机构             | `Lower_Layer_Open/Close()`；中层舵机第三代未使用 |

## 3. 双 HX711 称重设计

第三层称重板左右两侧各放置一套 HX711。

| 通道        | 用途     | SCK  | DT   |
| ----------- | -------- | ---- | ---- |
| HX711_LEFT  | 左侧称重 | PB10 | PB11 |
| HX711_RIGHT | 右侧称重 | PB14 | PB15 |

称重流程要求：

1. 进入称重状态时，5/6号丝杆推板停止不动，7号打包电机停止。
2. 等待 `weighTime = 1000 ms`，让大葱在称重板上稳定。
3. 左右两路 HX711 分别采样、去皮、换算重量。
4. 最终重量建议按两路相加：

```text
weight_total = weight_left + weight_right
```

5. 如果其中一路无有效采样，该路重量记为 0，并保留错误标记供调试显示。
6. 如果两路都无有效采样，最终重量为 0，避免沿用旧数据。
7. 称重完成后 OLED 显示 `weight_total`，并进入推板与打包流程。

## 4. 丝杆推板参数

5/6号电机驱动丝杆推板，将大葱从称重板推入旋转打包装置，打包完成后推板返回初始位置。

机械参数：

| 参数             | 数值   |
| ---------------- | ------ |
| 丝杆导程         | 8 mm   |
| 推动行程         | 300 mm |
| 位置模式目标脉冲 | 320000 |
| 推动速度         | 30 RPM |
| 返回速度         | 30 RPM |
| 加速度           | 0      |
| 推动方向         | CCW    |
| 返回方向         | CW     |

v1 到位判定：

- 暂不解析 CAN 电机到位反馈。
- 暂不使用限位开关。
- 使用时间估算作为推动/返回完成条件。
- 按 30 RPM、8mm 导程、300mm 行程估算，理论运动时间约 75 秒。
- 后续可增加安全余量，例如 `lowerPushTimeoutMs = 80000 ms`、`lowerReturnTimeoutMs = 80000 ms`。

## 5. 自动状态机

建议将第三代自动流程整理为以下状态：

```text
READY
UPPER_RUN
MIDDLE_RUN
MIDDLE_CUTTING
LOWER_WEIGH
LOWER_PUSH_AND_PACK
PACK_TAIL
LOWER_RETURN
WAIT_NEXT_CYCLE
STOP_UPPER
STOP_MIDDLE
STOP_LOWER
DONE
ERROR
```

### 5.1 READY

- 关闭切刀、水泵和舵机执行机构。
- 确认 5/6号推板处于初始位置，7号打包电机停止。
- 进入 `UPPER_RUN`。

### 5.2 UPPER_RUN

- 启动上层电机 1、2，速度模式同步运行。
- 默认速度 60 RPM。
- 运行 `runDuration = 5000 ms`。
- 时间到后停止 1、2号电机。
- 进入 `MIDDLE_RUN`。

### 5.3 MIDDLE_RUN

- 启动中层电机 3、4，速度模式同步运行。
- 默认速度 60 RPM。
- 打开下料执行机构：`Lower_Layer_Open()`。
- 等待 `startDelayMs = 500 ms`。
- 进入 `MIDDLE_CUTTING`。

### 5.4 MIDDLE_CUTTING

- 打开切刀：`Set_Relay_Switch(RELAY_CUTTER, 1)`。
- 中层继续运行或按实际机械需求保持当前状态。
- 持续 `runDuration = 5000 ms`。
- 时间到后停止 3、4号电机。
- 关闭切刀：`Set_Relay_Switch(RELAY_CUTTER, 0)`。
- 关闭下料执行机构：`Lower_Layer_Close()`。
- 大葱落到第三层称重板。
- 进入 `LOWER_WEIGH`。

### 5.5 LOWER_WEIGH

- 5、6、7号电机全部保持停止。
- 等待称重稳定时间 `weighTime = 1000 ms`。
- 读取左右两路 HX711，计算 `weight_total`。
- OLED 显示重量。
- 进入 `LOWER_PUSH_AND_PACK`。

### 5.6 LOWER_PUSH_AND_PACK

- 同时启动 5/6号丝杆推板和 7号打包电机。
- 5/6号电机使用位置模式：

```text
motor = 5, 6
mode = position
dir = CCW
vel = 30 RPM
acc = 0
clk = 320000
snF = true
```

- 5/6号位置命令下发后，通过广播同步启动。
- 7号电机使用速度模式：

```text
motor = 7
mode = velocity
dir = CCW
vel = 30 RPM
acc = 0
snF = true
```

- 状态持续到推板推动时间到达，例如 `lowerPushTimeoutMs = 80000 ms`。
- 时间到后停止 5、6号电机。
- 7号电机继续运行，进入 `PACK_TAIL`。

### 5.7 PACK_TAIL

- 7号打包电机继续运行 `packTailTimeMs = 3000 ms`。
- 时间到后停止 7号电机。
- 进入 `LOWER_RETURN`。

### 5.8 LOWER_RETURN

- 启动 5/6号丝杆推板返回初始位置。
- 5/6号电机使用位置模式：

```text
motor = 5, 6
mode = position
dir = CW
vel = 30 RPM
acc = 0
clk = 320000
snF = true
```

- 持续到返回时间到达，例如 `lowerReturnTimeoutMs = 80000 ms`。
- 时间到后停止 5、6号电机。
- 第三层流程结束，进入 `WAIT_NEXT_CYCLE`。

### 5.9 WAIT_NEXT_CYCLE

- 等待 `cycleDelayMs = 600 ms`。
- 回到 `UPPER_RUN`，开始下一轮。

## 6. 停机逻辑

### 6.1 优雅停机

收到 STOP 且当前处于自动模式时，执行优雅停机：

1. 进入 `STOP_UPPER`，停止上层 1、2号电机，不再进料。
2. 等待 `stopDelayMs = 5000 ms`，让已进入中层的物料继续处理。
3. 进入 `STOP_MIDDLE`，停止中层 3、4号电机，关闭切刀和下料执行机构。
4. 等待 `stopDelayMs = 5000 ms`，让第三层完成当前安全动作。
5. 进入 `STOP_LOWER`，停止 5、6号丝杆推板和 7号打包电机。
6. 关闭水泵、切刀、舵机执行机构。
7. 进入 `DONE`，回到 `MODE_IDLE`。

### 6.2 急停

收到急停命令后立即执行：

1. 广播停止 1-7号电机。
2. 广播失能 1-7号电机。
3. 停止 5/6号推板和 7号打包电机，不等待当前位置完成。
4. 关闭切刀、水泵和所有舵机执行机构。
5. 软件状态进入 `MODE_ERROR`。
6. 8号电机第三代不使用，不作为控制对象；如果仍接在总线上，广播停止会一并影响它。

## 7. 后续代码改造要点

1. `Conveyor_belt` 层需要把第三层从传送带抽象改为丝杆推板抽象。
2. 新增 5/6号位置模式同步控制接口，例如：

```c
bool StartLowerPush(void);
bool StartLowerReturn(void);
bool StopLowerScrew(void);
```

3. 打包只保留 7号电机接口，例如：

```c
bool StartPackMotor(float speedRpm);
bool StopPackMotor(void);
```

4. 移除第三代自动流程中对 8号电机的调度。
5. HX711 驱动需要支持两个独立通道。
6. `app_Conveyor` 状态机需要新增 `LOWER_PUSH_AND_PACK`、`PACK_TAIL`、`LOWER_RETURN` 等状态。
7. v1 使用时间估算推板到位；后续可升级为 CAN 到位反馈或限位开关。

## 8. 验证场景

1. 单独测试上层：1、2号电机同步运行 5 秒后停止。
2. 单独测试中层：3、4号电机同步运行，切刀按时打开和关闭。
3. 单独测试称重：左右两路 HX711 分别可读，合成重量正确显示。
4. 单独测试推板推动：5、6号 CCW 位置模式同步运动 320000 脉冲。
5. 单独测试推板返回：5、6号 CW 位置模式同步运动 320000 脉冲。
6. 单独测试打包：7号 CCW 速度模式 30 RPM，能启动和停止。
7. 联动测试：称重完成后，5/6号推动和7号打包同步启动；推板到位后7号继续运行 3000ms。
8. 自动循环测试：上层 -> 中层 -> 称重 -> 推动并打包 -> 返回 -> 下一轮。
9. STOP 测试：按优雅停机顺序依次停止上层、中层、第三层。
10. 急停测试：立即停止 1-7号电机和所有执行机构。
