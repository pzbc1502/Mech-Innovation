# STM32F407 三层洗菜系统外设接线引脚表

> 本文档按当前工程源码和 `STM32F407_UART_CMD.ioc` 整理，只记录现有配置。之前关于 USART1 PB6/PB7 串口屏的方案已作废，当前工程尚未给陶晶驰 X5 串口屏分配接口。

## 1. 必须接入的外设

| 外设            | STM32 引脚 | 工程接口                  | 接线说明                                                    |
| --------------- | ---------- | ------------------------- | ----------------------------------------------------------- |
| CAN 电机总线 RX | PB12       | CAN2_RX                   | 接 CAN 收发器 RXD，再接 Emm_V5 电机总线                     |
| CAN 电机总线 TX | PB13       | CAN2_TX                   | 接 CAN 收发器 TXD，再接 Emm_V5 电机总线                     |
| 左侧 HX711 SCK  | PB10       | HX711_LEFT_SCK            | 第三代称重板左侧 HX711 时钟                                 |
| 左侧 HX711 DT   | PB11       | HX711_LEFT_DT             | 第三代称重板左侧 HX711 数据                                 |
| 右侧 HX711 SCK  | PB14       | HX711_RIGHT_SCK           | 第三代称重板右侧 HX711 时钟，代码在 `HX711_Init()` 内配置 |
| 右侧 HX711 DT   | PB15       | HX711_RIGHT_DT            | 第三代称重板右侧 HX711 数据，代码在 `HX711_Init()` 内配置 |
| OLED SCL        | PB8        | 软件 I2C SCL              | OLED 使用软件 I2C，不走硬件 I2C 外设                        |
| OLED SDA        | PB9        | 软件 I2C SDA              | OLED 默认地址按 0x3C/写地址 0x78 使用                       |
| 切刀继电器      | PA5        | RELAY_CUTTER / RELAY_IN_1 | 继电器 1，控制切刀                                          |
| 水泵继电器      | PA6        | RELAY_PUMP / RELAY_IN_2   | 继电器 2，控制水泵                                          |
| 下层舵机 1      | PE9        | TIM1_CH1 / SERVO_CH1      | PWM 舵机输出                                                |
| 下层舵机 2      | PE11       | TIM1_CH2 / SERVO_CH2      | PWM 舵机输出                                                |

## 2. 电机编号与 CAN 总线

| 电机编号 | 当前用途             | 控制方式               | 接线说明                               |
| -------- | -------------------- | ---------------------- | -------------------------------------- |
| 1        | 上层传送带电机 1     | CAN2 / Emm_V5 速度模式 | 与 2 号同步运行                        |
| 2        | 上层传送带电机 2     | CAN2 / Emm_V5 速度模式 | 与 1 号同步运行                        |
| 3        | 中层传送带电机 1     | CAN2 / Emm_V5 速度模式 | 与 4 号同步运行                        |
| 4        | 中层传送带电机 2     | CAN2 / Emm_V5 速度模式 | 与 3 号同步运行                        |
| 5        | 第三层丝杆推板电机 1 | CAN2 / Emm_V5 位置模式 | 与 6 号同步推动/返回                   |
| 6        | 第三层丝杆推板电机 2 | CAN2 / Emm_V5 位置模式 | 与 5 号同步推动/返回                   |
| 7        | 旋转打包电机         | CAN2 / Emm_V5 速度模式 | 推板推动时同步启动，推板到位后继续运行 |
| 8        | 第三代不使用         | 不调度                 | 当前自动流程不接、不控                 |

CAN 接线注意：

- STM32 的 PB12/PB13 不能直接接 CANH/CANL，需要经过 CAN 收发器。
- CANH/CANL 需要共地，长线或多节点总线两端建议各接 120 欧终端电阻。
- 当前 CAN2 波特率按工程配置约为 500 kbps。

## 3. 人机输入与通信接口

| 外设          | STM32 引脚 | 工程接口  | 当前用途                             |
| ------------- | ---------- | --------- | ------------------------------------ |
| 按键 1        | PD4        | KEY_ONE   | START，启动自动流程                  |
| 按键 2        | PD5        | KEY_TWO   | RESUME/RESET，故障恢复               |
| 按键 3        | PD7        | KEY_THREE | STOP，优雅停机                       |
| 按键 4        | PD6        | KEY_FOUR  | EMERGENCY STOP，急停                 |
| Debug 串口 TX | PA9        | USART1_TX | `printf` 调试输出，115200 bps      |
| Debug 串口 RX | PA10       | USART1_RX | 调试输入，当前主要用于 USART1 初始化 |
| 蓝牙串口 TX   | PA0        | UART4_TX  | 蓝牙模块接收脚，115200 bps            |
| 蓝牙串口 RX   | PC11       | UART4_RX  | 蓝牙模块发送脚，115200 bps            |

按键接线注意：

- PD4/PD5/PD6/PD7 当前配置为上拉输入。
- 按键建议一端接对应 GPIO，一端接 GND，按下为低电平。

## 6. 上电前检查清单  

- 所有外设必须与 STM32 共地。
- 舵机建议单独供电，不要直接从 STM32 板载 3.3V 取大电流。
- 继电器模块电平逻辑需要上板确认；当前代码和注释里存在高低电平描述不完全一致的问题。
- CAN 电机驱动器编号需要设置为 1 到 7，8 号不要接入第三代自动流程。
- OLED、HX711、蓝牙等模块的供电电压要与模块规格一致，避免 5V 信号直接灌入不耐压的 STM32 引脚。
